# Shadow Server

Shadow Server é um projeto de servidor Node.js para gerenciamento de clientes, produtos e cobranças.

## Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/shadow_server.git
   cd shadow_server
npm install
npm start

Uso

    Acesse o servidor em http://localhost:3000.
    Use as rotas fornecidas para visualizar, adicionar, atualizar e excluir clientes, produtos e cobranças.

Estrutura do Projeto

    app.js: Arquivo principal do servidor Node.js.
    public/: Diretório para arquivos estáticos (HTML, CSS, JavaScript).
    src/: Diretório para código fonte do servidor.
        data/: Diretório para armazenamento de dados em arquivos JSON.
        routes/: Diretório para rotas do servidor.

Tecnologias Utilizadas

    Node.js
    Express
    npm
